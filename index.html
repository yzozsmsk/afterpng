Vektörel JSX
After Effects — Native vektör
  • Sonsuz ölçeklenebilir, kayıpsız
  • Native AE shape layer olarak açılır
  • Tüm keyframe'ler korunur
  • Renk, path, stroke AE'de düzenlenebilir
Nasıl kullanılır

Layers panelinde Vektörel AE export toggle'ını aç → Format olarak PNG seç → Export → _vectors.jsx indirilir → AE'de File → Scripts → Run Script File… → dosyayı seç → comp otomatik oluşur, tüm layer'lar native shape layer olarak gelir.

PNG + Script
After Effects & profesyonel
  • Layer bazlı düzenleme
  • Sıfır kalite kaybı, lossless
  • Şeffaf arka plan (alfa)
  • Otomatik AE import scripti
Nasıl kullanılır

ZIP'i çıkart → AE'de File → Scripts → Run Script File… → import_to_ae.jsx seç. Toplu export'ta batch_import.jsx tüm animasyonları bir kerede ekler.

MP4 / APNG
Video & sosyal medya
  • Yüksek kalite MP4 (20 Mbps)
  • Premiere, DaVinci, Final Cut
  • "None" seçince → şeffaf APNG
  • Tüm tarayıcılarda çalışır
Nasıl kullanılır

İndir → video editörüne sürükle. Şeffaf arka plan için arka plan rengini "None" seç → APNG çıkar.
